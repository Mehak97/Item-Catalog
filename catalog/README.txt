Item Catalog

This catalog contains the records of restaurant and the menu item provided by each restaurant.
This project make use of flask library to create server and use sqlalchemy to manage database.

Requirments:
The project uses Python3 to run the server.
The modules needed to run the project are:
a)SQLAlchemy
b)Flask Framework
c)OAuth2Client
d)Requests
e)httplib2

Steps:
Clone the Repository to your laptop.
Run database_setup.py file to create a database that contains the tables for restaurant name and its menu items.
Run lotsofmenus_user.py file to enter sample data to the database.
Run finalProject1.py file to start the server.
Now you can access the catalog on localhost:5000/ address.
